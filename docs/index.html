Key Metrics & Thresholds

Critical benchmarks for evaluating DEM simulation quality

⚡ Ionic Conductivity

SE-SE Coordination > 2.4 Percolation threshold
SE Percolation > 70% Good ion pathway
NCM-SE Interface > 80% Active material utilization

⚙ Mechanical Properties

Overlap Ratio < 3% Good (valid contact model)
Hertzian Correlation > 0.9 Valid contact model
Rolling Fraction 20–40% Proper jamming behavior

DEM Validity Assessment

Understanding what DEM simulations can and cannot predict for real electrodes

✅

Highly Valid

Topology Preserved
  • Contact network topology
  • Coordination number distribution
  • Force chain pathways
  • Fabric tensor (structural anisotropy)
  • Layer-wise uniformity (CV)
⚠

Reference Only

Use with Caution
  • Porosity (~35% sphere packing limit vs real <5%)
  • Packing density values
❌

Not Predictive

Do Not Use Directly
  • Absolute electrode density
  • Direct ionic conductivity values

💡 Key Insight

NCM (AM) network achieves 99.9% percolation — electron pathways are well-established. However, SE network shows 0% percolation in rigid-sphere DEM, because real SE particles undergo plastic deformation during calendering to form continuous ion pathways. DEM topology analysis remains valid for understanding relative contact quality and network structure.

📏 Scale Factor

Default scale: 1000× (simulation uses mm, real particles are μm)

Standard Mode

  • AM: 4 mm → 4 μm
  • SE: 0.8 mm → 0.8 μm

Bimodal Mode

  • AM_P: 4–5 mm → 4–5 μm
  • AM_S: 2–2.5 mm → 2–2.5 μm
  • SE: 0.5–0.8 mm → 0.5–0.8 μm
