연구 개요

목표

  • Halogen(Cl/Br) 치환이 argyrodite 기계적 물성에 미치는 영향 규명
  • DFT + MLIP multi-scale pipeline으로 B₀, E, Cij, Wad 예측

핵심 발견

  • Br↑ → B₀/E↓ (이온반경 + 분극성 + 전하밀도)
  • Li ordering → C44 47% 변동 (전단 민감)
  • Vacancy → chemical anchor → Wad↑

Vacancy → Chemical Anchor

  • Li₆ (vacancy 없음): 표면 Li 포화 → NCM과 추가 결합 없음 → Wad↓
  • Li₅.₄ (vacancy 있음): under-coordinated Li → NCM O²⁻ 당김 → Wad↑

v5 Method (진행 중)

  • Crystalline slab + surface-only MQA
  • SE 2×2×1 + NCM 5×5×1 (strain ~1%)
  • 5 z-cut sampling → vacancy 노출 통계
  • Isolated slab method for Wad

Pipeline v1 vs v2

v1 (현재 논문)

Halogen enumerate → MLIP screen → Rietveld Li (고정) → DFT relax → DFT EOS → Post-processing

  • Li 재최적화 없음
  • comp5 basin 문제: ΔC44=12.7 GPa
  • v108 basin 전환 빈번

v2 (권장, 검증 중)

Halogen enum → Halogen screen → Li screen (20개) → 500K annealing → MLIP EOS → DFT EOS

  • Li energy spread = 1162 meV 발견!
  • Ranking 역전 확인 (#15: 4위→2위)
  • Champion #0: -217.533 eV
  • MLIP B₀ = 26.9 (v1: 26.2) → Δ=0.7
